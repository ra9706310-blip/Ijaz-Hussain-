# 🏎️ Car Racing Game

A fun and interactive car racing game built with HTML5 Canvas and JavaScript!

## Features

- 🚗 **Player-controlled car** with smooth movement and acceleration
- 🛣️ **Dynamic road** with animated lane markings
- 🚨 **Enemy cars** spawning at increasing difficulty
- 📊 **Score tracking** as you avoid obstacles
- ⚡ **Speed display** showing current velocity
- 🎮 **Smooth controls** with keyboard input

## How to Play

1. Open `index.html` in your web browser
2. Use the arrow keys or WASD to control your car:
   - **← / A** - Move left
   - **→ / D** - Move right
   - **↑ / W** - Speed up
   - **↓ / S** - Slow down
3. Avoid the red enemy cars coming toward you
4. Each car you successfully pass adds 10 points to your score
5. Colliding with an enemy car ends the game
6. Press **R** to restart after game over

## Game Mechanics

- **Score System**: Gain 10 points for each enemy car that passes you
- **Difficulty**: Enemy spawn rate increases as your score increases
- **Physics**: Car has acceleration, friction, and maximum speed
- **Collision Detection**: Hit any red car to end the game

## Files

- `index.html` - Main game HTML structure
- `style.css` - Game styling and layout
- `game.js` - Game logic and mechanics

## Technical Details

Built with:
- HTML5 Canvas for rendering
- Vanilla JavaScript for game logic
- CSS3 for styling and animations
